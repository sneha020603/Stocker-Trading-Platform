# 📈 Stocker Trading Platform

A full-stack stock trading simulation platform built using Flask, MySQL, and yfinance API.

## 🚀 Features
- User authentication with OTP
- Buy/Sell stock simulation
- Portfolio tracking
- Real-time stock prices
- Admin dashboard
- Analytics dashboard

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MySQL
- API: yfinance

## ▶️ How to Run
1. Clone repo
2. Create virtual environment
3. Install requirements:
   pip install -r requirements.txt
4. Run:
   python app.py
